# Go Language

## Installation

```sh
#Easy way
sudo apt install golang-go
sudo apt install gccgo-go
```

```sh
#Hard way
export VERSION=1.14.6
export OS=linux
export ARCH=amd64
wget -O go.tar.gz https://golang.org/dl/go$VERSION.$OS-$ARCH.tar.gz
tar -C /usr/local -xzf go.tar.gz
rm -rf go.tar.gz
```
